# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Purpose

This repository is a curated reference collection of research papers and resources related to cluster headache (CH) and their treatment. It is not a software project — there are no build steps, tests, or code to run.

## Structure

- `papers/` — PDF research papers and journal articles
- `overview.md` — Summary/index of collected resources

## Working in This Repo

- When adding new papers, place PDFs in `papers/` and update `overview.md` with a citation and brief summary.
- Use the PDF reading capability to analyze paper contents when asked.
- Maintain accurate medical/scientific terminology — this is a clinical research reference collection.
